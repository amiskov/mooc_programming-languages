# Среда разработки Dr. Racket

Сокращения клавиш:

  • Cmd+E — показать/скрыть REPL.
  • Esc+p — предыдущая команда в REPL. Esc+n — следующая.
  • Cmd + i — отформатирова весь код в файле.

Есть плагин для Vim-режима. Устанавливается через ракетовский менеджер плагинов raco. Тут написано, как добавить путь к raco в систему.
